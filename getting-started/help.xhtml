Help

Before we start delving into CFRMIS you may want to know where to turn if you get stuck.

CFRMIS Help

Although CFRMIS has its own Help button on the toolbar this is not currently backed up with any documentation, clicking the button only launches a popup error message. This is why we have produced this guide.

This Guide

In the first instance you should refer to this guide - all of the parts of CFRMIS that you will need to use are explained here.

Troubleshooting

Several common issues and how to resolve them are described in our Troubleshooting section.

If you still need help

For help relating to how we add information to the database, such as best practice around naming conventions or what information to include and you can’t find an answer in the /Appendices/index then contact:

  • The Home Fire Safety Team for queries relating to domestic premises, or

  • The Fire Protection Administrator for queries relating to other buildings.

For technical issues with accessing or using CFRMIS contact the IT Service Desk.
