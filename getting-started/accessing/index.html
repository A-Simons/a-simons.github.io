Accessing CFRMIS

Cfrmis is a web-based application, which gives you a number of options for how you access it.

Note

CFRMIS is accessed using the same logon details you use to sign into Windows. It should pick this up automatically by seeing that you’re signed into whichever device you’re using to access CFRMIS, but if it does ask for a username and password just enter the same one you use to log into your device.

URL

The simplest way to access CFRMIS is by entering its URL directly into your browser’s address bar or omnibox.

FirePlace

CFRMIS can also be accessed via FirePlace. Open Fireplace and, under Links, press NFRS applications, then CFRMIS Web Access.

Bookmark

The quickest and easiest way to get to CFRMIS, especially if you’re going to be doing it often, is to create a bookmark in your preferred web browser. This will allow you get to CFRMIS in a single click once you have your web browser open.

Using either of the above two methods (or by clicking here) open CFRMIS. Then press Ctrl+D on your keyboard. Depending on your browser you will probably now be given the option to name your bookmark, we suggest you name it CFRMIS.

Note

At present CFRMIS will only provide limited functionality if accessed from a device which is not connected to an NFRS network. If you are working remotely and need to access CFRMIS this should be done by opening the Remote Desktop. Once connected to the remote desktop you can access CFRMIS using any of the above methods.
