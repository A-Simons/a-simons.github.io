How to use this guide

This guide is intended to serve as an introduction to CFRMIS for NFRS personnel who are new to using the tool, and as a reference for more experienced users.

The guide is broadly broken up into two main categories - explanations of how CFRMIS works as a tool and style guidelines to make sure that where we’re recording information in the CFRMIS database we are doing so in a way that ensures our recorded are accurate, detailed and consistent.

Using CFRMIS

Navigate to specific topics by using links down left hand side, or follow through the whole guide sequentially by using the Previous and Next buttons at the bottom of each page.

Appendices

The appendices contain style guides for how data is to be recorded in CFRMIS in different circumstances.
