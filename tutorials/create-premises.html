Creating a Premises Record

Warning

Before creating a new premises make sure that you have carried out a thorough check of the database - you could end up creating a duplicate of a premises with a slightly different name or address format. See Query Other Building.

Domestic Premises

Click on Domestic Premises.

Todo

Find out how Prevention do this

Other Buildings

Press Other Buildings. A Query Other Building tab will open.

Press New to change the query form into a New Other Building form.

Note

You can abandon this record at any time by pressing Cancel.

Use the gazetteer to ensure that our location information for this premises is as accurate as possible. See Gazetteer for more information.

Now enter information into as many fields as you can, following the principles defined in the Appendix A - Protection Guidance Notes relevant to the type of premises that you are creating a record of.

The fields highlighted in red are mandatory - you will not be able to save the premises record unless you provide values for these fields.

Note

Remember that you should add as much information as you can, but only enter accurate information! Do not guess - it is better to have an incomplete record than an inaccurate one.

Save the record by pressing Save.

A dialogue box will appear with the Premises ID which will be assigned to the new premises record and a confirmation request. Click Yes to continue.

The New Other Building tab will now be an Other Building tab containing the premises record you have just created.

Remember, if you had to choose a nearby premises from the gazetteer because the one you needed was not available now is the time to :doc`edit-map`.
