The Quick Access Buttons

The Quick Access Buttons provide quick access to the most commonly-used queries in CFRMIS. Depending on your User Profile’s permissions, you may see up to four buttons up here.

Other Buildings

Other Buildings is the menu that gives us access to buildings (and other premises) which are not used as a private residence.

Domestic Premises

Domestic Premises is the menu that gives us access to buildings and parts of buildings which are used as a private residence. This includes most houses, but also individual flats within large blocks

Clicking either of these buttons will launch a Premises Query tab in your Active Tabs Area. The Domestic Premises button will only allow you to see domestic premises, the Other Buildings button will allow you to see everything else.

These buttons also each work as dropdown menu - by clicking the small arrow on the right hand side of each button you can open a Query tab for almost all of the tables that you’ll ever need.

Letter History

This opens the Query Letter History tab. This can be used to search the database for letters saved within CFRMIS. Due to the way in which information has been recorded in the past this query is of limited use and is rarely used, but can be useful in some specific cases.

Multimedia Details

This opens the Query Multimedia Details tab. This can be used to search the database for multimedia saved within CFRMIS. Due to the way in which information has been recorded in the past this query is of limited use and is rarely used, but can be useful in some specific cases.

Sub-Site Premises

This opens the Query Sub-Site Premises tab. This tab is used to search the database for premises which are part of a Site/Sub-Site relationship. This appears at the bottom of the list in the Domestic Premises menu.
