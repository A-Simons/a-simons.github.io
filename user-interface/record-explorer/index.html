Record Explorer

The Record Explorer on the left hand side of the CFRMIS interface is a navigation tool to help you quickly access information related to the Premises or Jobs currently displayed in the Active Tabs.

It is an accordion-style menu - only Premises or Jobs can be open at any one time. Expanding one by pressing the will collapse the other.

When no Premises or Jobs are active the Record Explorer is displayed as above. Clicking on Domestic Premises will open a tab in the Active Tabs area displaying a list of all Domestic Properties on record. Similarly, clicking on Other Buildings will open a tab in the Active Tabs area displaying a list of all Other Buildings on record.

When a Premises entry is open in the Active Tabs its related records appear in a “tree” in the Record Explorer, in either the Domestic Premises or Other Buildings folder as appropriate. The unique ID number for the Premises is shown at the top of the tree and all of the possible types of records that can relate to that Premises are displayed below it. These are called nodes.

You may notice that the nodes correspond to the types of records that we saw in the Other Buildings and Domestic Premises menus when we looked at the Topbar.

It looks like a lot, but remember that many of these record types are not currently used within NFRS. In following releases of CFRMIS these may be hidden in order to provide a cleaner workspace.

Where there is an existing record related to a Premises then the table which holds the list of records of that type will have a green tick icon, and a number in brackets to indicate how many entries of that type are recorded in connection with that Premises.

Clicking on any of the Record Types in the tree will open a tab in the Active Tabs area of the CFRMIS window. This tab will list all of the Premises’ records of the type you clicked. In the example above, clicking CFS Jobs (6) would open a tab which lists the 6 records of CFS Jobs which are related to Premises 00112427, along with some basic details.

More than one Premises record can be open in your Active Tabs at once. If this is the case then there will be a separate tree for each Premises. It is good practice to collapse any trees which you are not using to help make sure that the Record Explorer is less cluttered. You can remove a Premises tree from the Record Explorer by closing all of the tabs relating to it in the Active Tabs area.

If you need to use more of your screen for your Main Screen area, you can hide the Record Explorer by pressing the icon. If you want it back, just press the icon.
