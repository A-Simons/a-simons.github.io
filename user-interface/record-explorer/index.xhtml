Record Explorer

The Record Explorer on the left hand side of the CFRMIS interface is a navigation tool to help you quickly access information related to the Premises or Jobs currently displayed in the Active Tabs. When no Premises or Jobs are active the Record Explorer is displayed as above. Clicking on Domestic Premises will open a tab in the Active Tabs area displaying a list of all Domestic Properties on record. Similarly, clicking on Other Buildings will open a tab in the Active Tabs area displaying a list of all Other Buildings on record.

When a Premises entry is open in the Active Tabs its related records appear in a “tree” in the Record Explorer, in either the Domestic Premises or Other Buildings folder as appropriate. The unique ID number for the Premises is shown at the top of the tree and all of the possible types of records that can relate to that Premises are displayed below it.

You may notice that these correspond to the types of records that we saw in the Other Buildings and Domestic Premises menus when we looked at the Topbar.

Many of these record types are not currently used within NFRS.

Where there is an existing record related to a Premises then the table which holds the list of records of that type will have a green tick icon, and a number in brackets to indicate how many entries of that type are recorded in connection with that Premises.
