HMOs

When creating or updating a HMO premises on CFRMIS the General style guidelines should be adhered to, other than as below.

Warning

This is an attempt to describe the current practice most represented in our database, it is not intended to be prescriptive going forward.

Occupier

This should simply be “HMO”.

Todo

Property Name

This is used for recording the landlord of the premises.
