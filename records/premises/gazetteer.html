GazetteerÔÉÅ

When Creating a Premises Record the gazetteer should be used in order to ensure that we are able to capture as much precise location information about the premises as possible, and support further audit activity.

The gazetteer will search through its own records for the information that has been entered in any of the following fields of the New Other Building or New Domestic Premises forms:

  • Occupier

  • Property Name

  • Property No.

  • Address Line 1

  • Address Line 2

  • Locality

  • Town

  • Postcode

  • Easting

  • Northing

  • OSAPR (in the future this may be TOID)

Because the way in which the gazetteer records information may not always be completely intuitive it is generally best to search using only one of these fields, of which Postcode tends to be the most accessible while still sufficiently specific.

Note

The gazetteer records all post codes as 7 character strings, which means that where you have a post code of the format XX00 0XX, like NN14 1UE, it should be typed into the Postcode field as NN141UE, without the space. As this is not how we record post codes throughout the rest of CFRMIS it is important that you remember to change it back to NN14 1UE after you have copied information over from the database.

Enter the postcode for the premises that you want to create a record for and click Gazetteer. The Gazetteer window will appear with a table of all properties matching your search criteria, in this case the postcode. Click to select the appropriate address for the premises you want the details for. Once you have clicked on a row of the table a map will appear in the pane to the right of the window. Use this to confirm that you have the right building.

Note

If the premises you are creating does not appear in the gazetteer table, perhaps because it is a new building, select the nearest available address for now and correct the address information after it has been copied into the form. Later you will need to manually adjust the coordinates in order to ensure that the location information on the premises record is as accurate as possible. See Using Edit Map.

At the bottom of this window are two tickboxes, Copy Prop. Name and Copy Occupier. One or both of these will be available for selection. Tick all boxes which are available and then click OK in the bottom right of the window. This will copy the premises information from the gazetteer into the new premises form.

Remember to add the space back into the postcode if you had to remove it earlier.
